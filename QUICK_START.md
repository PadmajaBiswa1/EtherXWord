# EtherX Quick Start Guide

## Getting Started in 3 Steps

### 1. Start the Development Server

```bash
cd frontend
npm run dev
```

The app will start on **http://localhost:5174** (or 5173 if available)

### 2. Open in Browser

Navigate to **http://localhost:5174** and you'll see the beautiful startup screen with:
- Blank document option
- Template cards
- Recent files (if any)

### 3. Start Creating!

Click **"Blank document"** to open the editor and start typing!

## Quick Feature Tour

### Basic Text Formatting

1. **Bold Text**: Select text → Click Bold button (or Ctrl+B)
2. **Italic**: Select text → Click Italic button (or Ctrl+I)
3. **Underline**: Select text → Click Underline button (or Ctrl+U)
4. **Change Font**: Click font dropdown → Select font
5. **Change Size**: Click size dropdown → Select size
6. **Text Color**: Click color palette icon → Pick color
7. **Highlight**: Click highlighter icon → Pick color

### Paragraph Formatting

1. **Alignment**: Click Left/Center/Right/Justify buttons
2. **Bullets**: Click Bullets button for unordered list
3. **Numbering**: Click Numbering button for ordered list
4. **Line Spacing**: Click Line Spacing dropdown → Select spacing

### Insert Content

1. **Table**: Go to Insert tab → Click Table button
2. **Image**: Go to Insert tab → Click Picture → Upload image
3. **Link**: Go to Insert tab → Click Link → Enter URL
4. **Symbol**: Go to Insert tab → Click Symbol → Pick symbol

### File Operations

1. **Save**: Click File menu → Save As (or Ctrl+S)
2. **Open**: Click File menu → Open → Select .html or .txt file
3. **Print**: Click File menu → Print (or Ctrl+P)
4. **New**: Click File menu → New

### View Options

1. **Zoom**: Use zoom slider in status bar (bottom right)
2. **Ruler**: Go to View tab → Check Ruler
3. **Navigation**: Go to View tab → Check Navigation Pane
4. **Word Count**: Go to Review tab → Click Word Count

## Tips & Tricks

### 1. Quick Styling with Styles Gallery
Instead of manually formatting headings, click a style from the Styles gallery in the Home tab.

### 2. Navigation Pane for Long Documents
Enable Navigation Pane (View tab) to see an outline of all headings and quickly jump to sections.

### 3. Find & Replace
Press Ctrl+F to open Find & Replace dialog and quickly locate or replace text.

### 4. Word Count
Check your progress anytime - word count is always visible in the status bar, or open the full dialog from Review tab.

### 5. Page Setup
Customize margins and page size from File → Page Setup or Layout tab → Margins.

### 6. Zoom Control
Use the zoom slider in the status bar or type a specific zoom level. Range: 10% - 500%

### 7. Comments
Add comments to collaborate: Review tab → New Comment. Comments appear in the right panel.

### 8. Recent Files
Your recently saved documents appear on the startup screen for quick access.

## Common Tasks

### Creating a Professional Letter

1. Start with blank document
2. Set margins: Layout → Margins → Narrow
3. Add your address at top
4. Format as you type
5. Save when done

### Creating a Resume

1. Click Resume template on startup screen
2. Or start blank and use Heading styles
3. Format with bullets and bold
4. Save as "My Resume.html"

### Inserting a Table

1. Insert tab → Table
2. Click to insert 3x3 table (default)
3. Click inside cells to type
4. Resize columns by dragging borders

### Adding Images

1. Insert tab → Picture
2. Select image from computer
3. Image appears inline in document
4. Resize by dragging corners (if needed)

### Creating Headers & Footers

1. Scroll to top/bottom of page
2. Click in header/footer area
3. Type your content
4. Apply formatting as needed

## Troubleshooting

### Document not saving?
- Check browser console for errors
- Ensure you have a filename
- Try a different browser

### Formatting not working?
- Make sure text is selected first
- Try refreshing the page
- Check if you're in the correct tab

### Images not uploading?
- Check file size (keep under 5MB)
- Supported formats: JPG, PNG, GIF
- Try a different image

### Zoom too small/large?
- Use zoom slider in status bar
- Or set to 100% for normal view
- Minimum: 10%, Maximum: 500%
